# MAC2IPv6

Usage python3 mac2ipv6.py <MAC ADDRESS>

```
 $ python3 mac2ipv6.py 52:74:f2:b1:a8:7f                                                                                                                                  
[*] MAC ADDRESS: 52:74:f2:b1:a8:7f
[*] IPv6: fe80::5074:f2ff:feb1:a87f
```
